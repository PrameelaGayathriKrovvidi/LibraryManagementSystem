# LibraryManagementSystem
Library Book Reservation System is a Core Java console application that manages books and user reservations using OOP principles. It enforces business rules like preventing duplicate reservations and handles invalid inputs with custom exceptions, showcasing clean design, state management, and Java Collections usage.

**Key Objectives**

    Manage books and users efficiently
    
    Prevent reserving already reserved books
    
    Handle invalid inputs gracefully
    
    Apply OOP principles and clean architecture
    
    Simulate real-world business logic using Core Java
